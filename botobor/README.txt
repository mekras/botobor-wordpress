== Description ==

Protect forms from spam using [Botobor](https://github.com/mekras/botobor/) library.

== Installation ==

1. Extract `botobor` folder from archive.
2. Upload the entire `botobor` folder to the `/wp-content/plugins/` path on your site.
3. Activate the plugin through the 'Plugins' screen in WordPress/
